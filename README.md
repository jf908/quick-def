# Quick Def

A web application for keeping track of word definitions.

![Screenshot](https://i.imgur.com/ROMRxvR.png)

## Prerequisites

Node v10+

## Setup

```
npm install
```

## Running

```
npm start
```

Open up http://localhost:5678

## Built With

* [Restana](https://www.npmjs.com/package/restana)
* [Vue](https://vuejs.org/)
